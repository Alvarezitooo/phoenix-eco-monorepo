## 🚀 **INSTALLATION ÉTAPE PAR ÉTAPE**

### **ÉTAPE 1 : Ollama (5 min)**
```bash
# Installation Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# Vérification
ollama --version

# Démarrage service
ollama serve &
```
